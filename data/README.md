# Data Directory

Place your competition data files here:

## Required Files

- `train.csv` - Training data with SalePrice
- `test.csv` - Test data for predictions
- `data_description.txt` - Feature descriptions

## Download

Download from the [Kaggle competition page](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data).

## Data Structure

**Training Set:** 1,460 houses × 81 columns (79 features + Id + SalePrice)

**Test Set:** 1,459 houses × 80 columns (79 features + Id)

## Important Notes

- CSV files are NOT tracked in git (see `.gitignore`)
- Make sure files are in this directory before running scripts
- Do not commit data files to the repository
